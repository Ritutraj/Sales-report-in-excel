# Sales-report-in-excel
Sales report of Atliq hardware describing customer performance, market performance and profit &amp; loss . 
Atliq is a company that sells hardware to different customers which are stores or online platforms and they inturn sells them to end customers. 4 excel files were given in which 3 were dimension table and 1 fact table. Atliq's requirement was to build a customer performance table, market performance table and profit & loss table. Detailed report is done in excel.
